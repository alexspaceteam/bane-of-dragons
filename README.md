# Bane of Dragons

A minimal Paper plugin for Minecraft 1.21.1.

## Features

- Broadcasts a message to all players when someone joins the server
- Allows whitelisted players to launch a **dragon fireball** by right-clicking with a netherite sword

## Dragon Fireball


**How to use:** Hold a netherite sword and right-click. A dragon fireball launches in the direction you're looking.

## Requirements

- [Paper](https://papermc.io/) 1.21.1
- Java 21

## Building

```bash
./gradlew build
```

Output JAR: `build/libs/BaneOfDragons.jar`

## Installation

Drop `BaneOfDragons.jar` into your server's `plugins/` folder and restart.
